## ⚙️ How It Works

ShadowPack mixes game logic, blockchain flow, and AI chat behavior.

The game loop is simple:

1. You join a match.
2. The app assigns roles.
3. Six AI personalities talk, accuse, defend, and vote.
4. The game tracks each round on OneChain.
5. The match ends when one side wins.

The AI agents give each match a different feel. Some agents may act calm and careful. Others may push hard, lie, or react fast. That makes each round feel less predictable.

## 🧠 Main Features

### 🤖 Six AI Personalities
Each match uses six LLM-powered personalities. They do not act the same way, so the table feels more human.

### 🕵️ Social Deduction Gameplay
You must watch speech, voting, and blame patterns. Pick out who sounds honest and who sounds false.

### ⛓️ OneChain Support
The game uses OneChain for on-chain game flow. This helps keep match actions tied to the chain model used by the app.

### 💬 Text-Based Interaction
The game focuses on reading and choosing. That makes it easy to follow even if you are new to this genre.

### 🧱 Built With Modern Web Tools
ShadowPack uses TypeScript and Next.js. These help the app stay fast and organized.

## 🧭 How to Play

1. Start a new match.
2. Read your role and the first game prompt.
3. Watch each AI agent speak during the day phase.
4. Make your vote based on patterns, not one line alone.
5. Survive the night if you are a villager.
6. If you are werewolf, use your cover and mislead the group.
7. Keep playing until one team wins.

A good rule is to track who changes their story and who pushes votes too early.
